## 🚀 Kurulum ve Çalıştırma

### Gereksinimler
- Node.js (v16+)
- MongoDB
- npm veya yarn

### Hızlı Başlatma
```bash
# 1. Projeyi klonlayın
git clone <repository-url>
cd baybar-corporate

# 2. Backend'i başlatın
cd server
npm install
# .env dosyasını düzenleyin
npm run dev

# 3. Frontend'i başlatın (yeni terminal)
cd client
npm install
# .env dosyasını düzenleyin
npm run dev

# 4. Veritabanını seed edin
cd server
npm run seed
```

### Otomatik Başlatma
```bash
# Windows için
start.bat

# Veya manuel olarak
# Backend: http://localhost:5000
# Frontend: http://localhost:3000
# Admin: http://localhost:3000/admin
```
